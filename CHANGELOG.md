# Changelog

## 0.1.1 - 2026-05-03

- Refined the visual analysis panel with a wider dark-theme layout, improved player load cards, updated court map styling, and cleaner speed chart presentation.
- Added long-run processing progress reporting, including frame progress, percentage complete, average frame time, and estimated remaining minutes.
- Added clearer completion summaries for configured match sources so long batch runs end with a concise status report.
- Simplified shuttle-speed overlay behavior so the on-frame speed badge only appears when valid speed data is available.

## 0.1.0 - 2026-05-02

- Added portable project-relative path handling.
- Added setup scripts for Windows and Unix-like systems.
- Added model asset download and install health-check scripts.
- Added bilingual documentation and deployment notes.
- Added FFmpeg encoder fallback from NVIDIA NVENC to `libx264`.
